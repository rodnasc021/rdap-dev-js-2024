# rdap-dev-js-2024
Aula prática de Desenvolvimento em JavaScript pela faculdade Anhanguera.  Nessa aula eu aprendi a criar um projeto no PlayCode.io, usando HTML, CSS e JavaScript. O projeto era composto por um pequeno formulário com campo para corpo de e-mail que deve ser preenchido pelo usuário, onde ele verifica se ele foi inserido da forma correta.
